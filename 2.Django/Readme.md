# Proyecto Django

## Create Virtualenv
```sh
$ virtualenv .venv
$ source .venv/bin/activate
```

## Install dependencies (OPCIONAL: si no existe la DB)
```sh
$ make init
```

## Run Server
```sh
$ make run
```
